# Replication for the paper "Street green space and electricity demand: evidence from metered consumption data" (Falchetta, De Cian, Lunghi, 2025)
Replication codes for Giacomo Falchetta, Enrica De Cian, Jacopo Lunghi, "**Street green space and electricity demand: evidence from metered consumption data**" \[Energy Economics\]

[giacomo.falchetta@cmcc.it](mailto:giacomo.falchetta@cmcc.it)

# Description
This repository provides the codes required to reproduce the tables, figures, and in-text summary statistics in the paper. 

The R scripts "1_main.R" and "2_ancillary.R" replicate the core empirical analysis, output tables, and figures presented in the manuscript. Additional R scripts produce figures and tables contained in the Appendix or additional data descriptive items. To replicate the analysis, adjust the directory paths of the R scripts to the local path of cloned repository. Then, run the script. 

## Data for replications
Please note that the electricity consumption microdata is confidential and cannot be shared due to limited rights of sharing the dat. Please contact the authors for access requests, which will be evaluated on a per-case basis.
